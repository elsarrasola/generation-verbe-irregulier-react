Générateur d'évaluation de verbes irréguliers

Sélectionnez les verbes à évaluer et les colonnes que vous souhaitez afficher. Vous pouvez également choisir le nombre de trous à remplir pour chaque ligne.
Ces trous seront répartis de manière aléatoire parmi les colonnes sélectionnées.
Il est également possible de mélanger l'ordre des verbes.

Note : Le document ne sera pas rendu dans ce style "pixélisé" [c'est le développeur qui se régale.]

PS : Bon courage pour la correction !
